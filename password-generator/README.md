# PASSWORD GENERATOR

A simple React application that generates secure passwords based on user preferences.

FEATURES
.generates random password instantly
.set password lenght (8-32)
.user can include numbers and special characters to the password
.user can simply copy the generated password to clipboard
.responsive and simple ui

TECHNOLOGIES USED
.React
.javaScript
.Tailwind CSS


INSTALLATION
    1. Clone the repo : git clone <repo-link>
    2. Navigate to the project folder : cd password-generator
    3. Install dependencies : npm i/install
    4. Run the project : npm run dev

USAGE
    1. Adjust the password length using slider
    2. Enable or disable numbers and special characters
    3. A password will be generated automatically
    4. Click the Copy button to copy the password
